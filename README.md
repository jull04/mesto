# Проект: Место
Ссылка на проект: https://jull04.github.io/mesto/


## Описание проекта

 Сервис Mesto: интерактивная страница, куда можно добавлять фотографии, удалять их и ставить лайки.

## Используемые технологии

* JavaScript, HTML, CSS
* флексбокс-вёрстка
* Grid Layout
* адаптивная верстка
* БЭМ
